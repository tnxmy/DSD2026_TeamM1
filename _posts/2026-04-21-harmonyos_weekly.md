## 📖 本周记录 / This Week's Notes

在这周，我对鸿蒙系统的UI开发有了一次尝试，首先我们在这个网址上下载集成开发环境，https://developer.huawei.com/consumer/cn/deveco-studio/ 。对于UI开发，华为官网上有实例代码 https://gitcode.com/HarmonyOS_Samples/HarmonyOSComponentUXExamples ，里面包含各种容器和组件，然后我简单的完成了一个app，在这周里，我们会设计好一个初步完整的app的页面逻辑，并且编写代码。

> *This week, I made an attempt at UI development for the HarmonyOS system. First, we downloaded the integrated development environment from this website, https://developer.huawei.com/consumer/cn/deveco-studio/ For UI development, there are example codes available on Huawei's official website https://gitcode.com/HarmonyOS_Samples/HarmonyOSComponentUXExamples It contains various containers and components, and I have simply completed an app. This week, we will design a preliminary and complete app page logic and write code.*
